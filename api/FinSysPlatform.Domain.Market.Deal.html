IsEntry

This execution is considered as entry (position before was Flat or the execution increased current position by adding new quantity or the execution changed direction of position (exit+entry))

public bool IsEntry { get; }

Property Value

bool

Examples

Ex 1: Position before the execution was Flat, this execution is Sell 1. Result position is Short 1. IsEntry=true (entry for flat position) Ex 2: Position before the execution was Long 1, this execution is Long 1. Result position is Long 2. IsEntry=true (adding to current position) Ex 4: Position before the execution was Long 1, this execution is Sell 2. Result position is Short 1. IsExit=true, IsEntry=true (position changed direction)

IsExit

This execution is considered as exit (the execution has a direction against current position (Sell vs Long, Buy vs Short) which leads to position reduce) Exit may be full (result position flattens) or partial (result position is not flat). In any case, this is exit

public bool IsExit { get; }

Property Value

bool

Examples

Ex 1: Position before the execution was Long 1, this execution is Sell 1. Result position is Flat. IsExit=true (full exit) Ex 2: Position before the execution was Long 2, this execution is Sell 1. Result position is Long 1. IsExit=true (partial exit) Ex 3: Position before the execution was Long 1, this execution is Buy 1. Result position is Long 2. IsExit=false (position increased) Ex 4: Position before the execution was Long 1, this execution is Sell 2. Result position is Short 1. IsExit=true, IsEntry=true (position changed direction)
