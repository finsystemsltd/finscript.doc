Remarks

To calculate a margin for XXX/YYY instrument we have

  1. To calculate pure margin: margin = quantity / Leverage
  2. To convert it to base currency (BaseCurrency). If base currency == XXX, then return pure margin. If base currency == YYY, then return pure margin * current currency rate (price).
  3. If base currency is neither XXX nor YYY, we have to calculate the rate via cross-rates. See details TryGetRate(string, string, out double)
