DefaultStopLossPercent

To calculate the quantity available to risk, we need to know the protective stop loss value. The stop loss should come as a parameter in StopLoss. But since this parameter is optional, it may absent. Then we need to provide some way to calculate the stop loss value by default. The simplest and most common way is to calculate the stop loss as a percentage of the current price. For example, if the current price is $100, and the default stop loss is 5%, then the stop loss will be $95 for Buy and $105 for Sell.

[UnitProperty(Name = "Default Stop Loss, %", CategoryName = "Parameters", Description = "Stop loss value calculated as % of current price. If stop loss is not specified explicitly, this value will be used to calculate stop loss", Order = 1)]
[Range(0.01, 10000)]
public double DefaultStopLossPercent { get; set; }
