Remarks

  1. Please note that, some properties do not have FinSysPlatform.NetExtension.Attributes.VisibleByDefaultAttribute, as those columns should not be visible by default on the UI.
  2. If you want to exclude some properties from your grid, then you should inherit your own grid representer and filter properties before sending the default collection to base constructor (see example)

Pay attention: the class supports IQuantityRepresenterFormatSettingsChangeEventHandler, and it does not subscribe to Changed global event directly If you need to refresh the item on format changes, you need to subscribe to Changed event on your own and then call OnQuantityRepresenterFormatSettingsChanged() method for each item
