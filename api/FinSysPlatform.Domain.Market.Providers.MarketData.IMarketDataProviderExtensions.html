GetLatestMarketData(IMarketDataProvider, Instrument, bool, bool, bool, int, Action<MarketDataValue>)

Returns latest known market data for specified instrument. If there is no current market data, the routine will wait (in synchronous mode) for data for specified timeoutMs milliseconds If time is out and still no data, callback will not be called. Pay attention, that callback will be called several times, for each bid, ask and last price types (which are enabled in parameters)

public static void GetLatestMarketData(this IMarketDataProvider provider, Instrument instrument, bool waitForBid, bool waitForAsk, bool waitForLast, int timeoutMs, Action<MarketDataValue> callback)
