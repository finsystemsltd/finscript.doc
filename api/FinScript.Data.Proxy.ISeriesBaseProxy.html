Remarks

Script units always have their input data series as non-nullable. However, a script unit gets its input later after creation when it is attached to a chart, another script unit, or any other data source. To provide non-nullable input series, the proxy is used. It is created with a nullable source series (Source) and can be assigned later. ISeriesBaseProxy inherits from ISeriesBase and forwards all calls to the source series (Source) when it is assigned, and returns empty values when the source is not assigned. This ensures that the script unit can operate without null checks, simplifying the code and reducing the risk of null reference exceptions.
