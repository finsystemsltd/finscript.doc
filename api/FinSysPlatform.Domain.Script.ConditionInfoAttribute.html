Properties

IsSeriesNeeded

The property is true if the condition needs a series of values. For example, "Is rising" condition means we need at least 2 points in the series to calculate False means the condition needs only a single value to calculate. For example "Equals to" condition means we need 1 point of the series and one parameter (One) to compare
