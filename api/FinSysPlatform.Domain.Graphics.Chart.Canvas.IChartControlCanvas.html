CreateBlit(string, Color, double?, string?, FontWeight?, FontStyle?, IRenderedTextBlit?)

Draws the specified text onto the bitmap. Using a bitmap allows to optimize drawing process (text once draws on the bitmap and then copies to the canvas)

IRenderedTextBlit CreateBlit(string text, Color color, double? fontSize = null, string? fontName = null, Font.FontWeight? fontWeight = null, Font.FontStyle? fontStyle = null, IRenderedTextBlit? currentBlit = null)

Parameters

Text color

currentBlit IRenderedTextBlit

Already existing blit to re-use. If all its parameters are equal to the specified parameters, it will be returned instead of creating new one

Returns

IRenderedTextBlit

Remarks

Using prepared bitmaps instead of DrawText(string, Point, Color, HorizontalOrigin, VerticalOrigin, double?, string?, FontWeight?, FontStyle?, IRenderBrush?) for static text is much more efficient. If your text parameters (text, color, font) are constant it is highly recommended to create a bitmap once and then to use it for drawing.
