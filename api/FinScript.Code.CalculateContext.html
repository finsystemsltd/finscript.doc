CurrentBarIndex

Current bar index which is calculated. Pay attention that CurrentBarIndex is not equal to ISeriesMasterSource.Count This index starts from 0 and ends with ISeriesMasterSource.Count to calculate each value of data

public int CurrentBarIndex { get; }

Property Value

int

FirstIndexInCalculationSession

First bar index in current calculation session When calculation is undergoing for a bunch of bars (for example, first calculation session runs calculation from first up to the last bar) first and last indexes of a calculation session may help to optimize calculation algorithm When unit runs in real-time mode it processes tick by tick and usually CurrentBarIndex = FirstIndexInCalculationSession = LastIndexInCalculationSession

public int FirstIndexInCalculationSession { get; }
