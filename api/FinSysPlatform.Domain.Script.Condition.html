[ConditionInfo(true, ConditionParameterCount.One)] CrossesAbove = 5

Series of 2 points crosses some level up: the previous data is below or equal to the level, the next data is above the level Input data: a) series of 2 values, b) some value (level) to compare

The condition is met when 2 points of the series crosses the specified level from below to above

[ConditionInfo(true, ConditionParameterCount.One)] CrossesBelow = 6

Series of 2 points crosses some level down: the previous data is above or equal to the level, the next data is below the level Input data: a) series of 2 values, b) some value (level) to compare

The condition is met when 2 points of the series crosses the specified level from above to below

[ConditionInfo(true, ConditionParameterCount.One)] FallsFromAbove = 8

Series falls from the peak that is above some value Input data: a) series of values of 3 items length, b) some value (level) to compare

The condition contains of 2 conditions: a) the series falls from the peak b) the detected peak is above the specified level To calculate the condition, we need 3 points of the series (the current point, the previous point that is supposed to be a peak, and pre-previous point to detect a peak) and some level You can also simplify the condition and remove part b) by setting the level to MinValue value

[ConditionInfo(false, ConditionParameterCount.One)] IsAbove = 0

Series is falling Input data: series of values of 3 items length

To calculate the condition, we need 3 points of the series (the current point, the previous point and pre-previous point), all points should descend from left to right

[ConditionInfo(false, ConditionParameterCount.Zero)] IsNotEmpty = 13

Series' value exists (not empty) IsValidValueAt(int)

[ConditionInfo(true, ConditionParameterCount.Zero)] IsRising = 10

Series is rising Input data: series of values of 3 items length

To calculate the condition, we need 3 points of the series (the current point, the previous point and pre-previous point), all points should ascend from left to right

[ConditionInfo(true, ConditionParameterCount.One)] RisesFromBelow = 7

Series' value rises from the valley that is below some value Input data: a) series of values of 3 items length, b) some value (level) to compare

The condition contains of 2 conditions: a) the series raises from the valley b) the detected valley is below the specified level To calculate the condition, we need 3 points of the series (the current point, the previous point that is supposed to be a valley, and pre-previous point to detect a valley) and some level You can also simplify the condition and remove part b) by setting the level to MaxValue value

Remarks

Each enum value has to have an explicitly defined number to for out of the app scopes tasks: exchanges, storage, etc.
