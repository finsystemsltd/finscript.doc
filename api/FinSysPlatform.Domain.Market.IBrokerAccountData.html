Margin

Currently locked by open positions money Margin depends on Leverage. For calculating margin you have to divide all your open positions by Leverage

double Margin { get; }

Property Value

double

Examples

Suppose we have open position long EUR/USD 1 Lot (100 000) at 1.14720 Without leverage (1:1) we need margin = $114 720. Now suppose we have Leverage = 100. Then margin = $1147.2

Remarks

see for details: https://www.investopedia.com/articles/forex/07/forex_leverage.asp https://admiralmarkets.com/education/articles/forex-basics/margin-in-forex-trading-margin-level-vs-margin-call

MarginPercent

Margin percent is Equity/Margin*100%. Can be NaN if information is not available Equals to PositiveInfinity if current margin =0 (no margin)

double MarginPercent { get; }

Property Value

double

Examples

Input values: Long EUR/USD 0.1 Lots (1000) at 1.09709 Leverage = 1:1 Equity = $5000 Calculated values: Margin = 1.097091000 = $1097 Margin percent = 5000/1097100=455%
