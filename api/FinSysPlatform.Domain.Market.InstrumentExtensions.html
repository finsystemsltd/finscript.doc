IsThis(Instrument?, Instrument?)

Compares references with ReferenceEquals(object, object) method Returns true if the passed instrument is the same instance as the current instrument or both references are null

Parameters

me Instrument
instrument Instrument

Returns

Remarks

Pay attention, both instruments can be null. In this case, the method returns true

There are 3 ways to compare instruments:

  1. IsSameKey(Instrument, Instrument?) - compares UniqueKey only
  2. ParametersEqual(Instrument, bool) - compares all parameters of the instrument
  3. IsThis(Instrument?, Instrument?) - compares references to instances
