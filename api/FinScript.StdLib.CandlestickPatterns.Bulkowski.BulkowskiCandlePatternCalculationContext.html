IsLowerShadowSmallEnough(double)

Checks if the lower shadow is small enough Many patterns use the condition: "no lower shadow or very little lower shadow"

public bool IsLowerShadowSmallEnough(double maxPossibleFactor)

Parameters

maxPossibleFactor double

The share of the CandleHeight to consider the shadow is small. For example, 0.1 means shadow should be less than 10% of full candlestick height (High-Low) The parameter should be greater than zero

Returns

bool

Remarks

If the candle has zero height (High==Low) this method always returns false
