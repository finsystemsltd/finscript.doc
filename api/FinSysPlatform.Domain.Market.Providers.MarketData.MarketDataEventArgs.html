Examples

Consider an example of market data provider that generates Bid, Ask and Last price updates separately.

  1. Bid price update: 100.00, Ask = NoValue, Last = NoValue: CurrentPrices = {100.00, NoValue, NoValue}
  2. Ask price update: 100.02, Bid = NoValue, Last = NoValue: CurrentPrices = {100.00, 100.02, NoValue}
  3. Last price update: 100.01, Bid = NoValue, Ask = NoValue: CurrentPrices = {100.00, 100.02, 100.01}
  4. Bid price update: 100.05, Ask = NoValue, Last = NoValue: CurrentPrices = {100.05, 100.02, 100.01}
  5. Ask price update: 100.06, Bid = NoValue, Last = NoValue: CurrentPrices = {100.05, 100.06, 100.01} ...

Remarks

Market data provider may generate Bid, Ask and Last price updates separately. For example, first - Bid info, then Ask info, then Last info. In this case, one packet of data does not contain info about each price. This property contains info about all prices: if the packet does not contain update for the price, the previous value is used.
