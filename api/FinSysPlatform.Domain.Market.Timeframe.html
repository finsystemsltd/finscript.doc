Fields

[ShortName("D")] Day = 4

Day. With this timeframe any time value should be aligned to a day, without time part, for example: 2019.12.28

[ShortName("H")] Hour = 3

Hour. With this timeframe any time value should be aligned to a hour, for example: 23:00:00

[ShortName("M")] Minute = 2

Minute. With this timeframe any time value should be aligned to a minute, for example: 23:53:00

[ShortName("MN")] Month = 6

Month. With this timeframe any time value should be aligned to first day of the month, for example: 2019.12.01

[ShortName("S")] Second = 1

Second. With this timeframe any time value should be aligned to a second, for example: 23:53:12,0000

[ShortName("T")] Tick = 0

Tick

This timeframe is not time-based. It means that time values (for example Time) is not aligned to any time border

[ShortName("W")] Week = 5

Week. With this timeframe any time value should be aligned to Sunday , for example: 2019.12.29 (this is Sunday)

[ShortName("Y")] Year = 7

Year. With this timeframe any time value should be aligned to first day of the year, for example: 2020.01.01

Remarks

All timeframes except Tick are time-based. It means that time values for those timeframes must be aligned Methods to align times:

  • ToSeconds(Timeframe, int)
  • AlignDateTime(Timeframe, DateTime)
  • IsTimeAligned(Timeframe, DateTime)
  • GetNextTime(Timeframe, DateTime)
  • GetPriorTime(Timeframe, DateTime)
