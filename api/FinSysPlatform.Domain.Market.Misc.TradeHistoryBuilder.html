CreateWithRestoredHistory(IBrokerageProvider, Instrument, IEnumerable<Execution>, Position, out Execution?)

Creates TradeHistoryBuilder with restored initial execution Often the execution history is not full and contains only last executions. Without knowing the start execution it is impossible to build the whole trade chain, but, knowing the final result (result position) it is possible to emulate initial execution to make the whole chain produce at the finish the current position. Emulated execution (if it was generated) always has index 0 and parameters: Time = MinValue Price = 0 ExecutionId = Restored Description = Restored Exchange = null
