Examples

Let's consider we have EUR/USD = 1.13, GBP/USD = 1.32. Having these rates, the calculator can calculate EUR/GBP rate (route: EUR->USD->GBP = 1.13/1.32 = 0.85) Another example: we have GOLD = 1783.68 USD, EUR/USD = 1.13. The calculator can calculate GOLD vs EUR rate (route: GOLD->USD->EUR = 1783.68/1.13 = 1.58) Calculation route can be more complex. For example, we have EUR/USD, EUR/CAD, GOLD/USD and we want to calculate rate GOLD/CAD: GOLD->USD->EUR->CAD
