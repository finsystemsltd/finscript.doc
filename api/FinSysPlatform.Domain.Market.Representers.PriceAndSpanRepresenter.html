Remarks

Unlike many other representers, this one has multiple values to show (current price and delta, and initial price needed for calculation) So, there is a question how to implement comparison. Since the FinSysPlatform.Domain.Market.Representers.PriceAndSpanRepresenter.CurrentPrice seems like a main value, we use it for comparison
