Remarks

The item has several market-dependent properties that must be updated regularly as fresh market data comes:

  • Profit
  • ProfitPoints
  • CurrentPrice This basic implementation of item does not support automatic refresh. It should implemented outside. To update properties, the method RefreshMarketDependentProperties() must be called from outside

Pay attention this class is not inherited from BaseGridRepresenterIndexedItem as many others, because it is updateable and supports INotifyPropertyChanged notification (via FinSysPlatform.NetExtension.Misc.NotificationObject) Pay also attention, that unlike many other representers, this one supports an updating of source position (Position) Since it cannot inherit VisualIndex, it implements VisualIndex from the scratch

Pay attention: the class supports IQuantityRepresenterFormatSettingsChangeEventHandler, and it does not subscribe to Changed global event directly If you need to refresh the item on format changes, you need to subscribe to Changed event on your own and then call OnQuantityRepresenterFormatSettingsChanged() method for each item
