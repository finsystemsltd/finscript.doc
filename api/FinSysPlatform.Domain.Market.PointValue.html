Remarks

Usually, for Forex instruments, point value changes constantly along with currency rates Some forex instruments (crosses like EUR/GBP, GBP/AUD etc) need rates of basic instruments (EUR/USD, GBP/USD etc) to calculate point value If those rates are not accessible, the point value can be calculated approximately
