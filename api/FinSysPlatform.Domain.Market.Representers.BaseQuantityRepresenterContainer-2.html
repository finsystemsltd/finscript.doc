Remarks

The main purpose of the class is to provide settings change tracking (QuantityRepresenterFormatSettings) As soon as the format settings change, the container recreates inner representer Value and fires PropertyChanged event Also, you can recreate representer manually by calling RecreateValue() method. For example, if the quantity value changes, you can recreate representer to update the view

Important: representer container subscribes to Changed event. it may lead to performance issue: representer containers may exist by thousands, and each of them will subscribe to the global event, which will lead to huge subscribers list and performance degradation and memory consumption. Do not use this class for numerous instances, like records in grids and so on
