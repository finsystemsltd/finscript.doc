GetAmountOfMoneyForInstrument()

Computes amount of money (converted to the base currency of the specified instrument) available for trade using specified allocation model.

protected double GetAmountOfMoneyForInstrument()

Returns

double

Examples

  1. Calculation type = Balance, Instrument = XXX/USD, Account = USD, Current balance = 300 USD Computation: Since the account currency is the same as the instrument currency, the rate 1:1. The result amount of money = 300 USD * 1 = 300 USD
  2. Calculation type = Balance, Instrument = XXX/EUR, Account = USD, Current balance = 300 USD Computation: the EUR/USD rate is 1.05. The result amount of money = 300 USD / 1.05 = 286 EUR

Remarks

The method requires Account to be set and brokerage provider BrokerageProvider avalable. The method multiplies the amount of money by the leverage of the instrument.
