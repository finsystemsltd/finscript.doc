A point-like chart object, defined by a single (Time, Value) coordinate There are several standard figures (PointType) available. Also, you can set the custom figure by using SetCustomShape(Point[])

SetCustomShape(Point[])

Sets the custom shape. The custom shape should be defined as a polygonal object with coordinates in ranges [0..1] In order to get back to standard shape, re-assign PointType

public void SetCustomShape(Point[] polygon)

Parameters

polygon Point[]

Remarks

For example, a full size square can be defined as (0,0)-(0,1)-(1,1)-(1,0), a up-directed triangle can be defined as (0.5, 0)-(1,1)-(0,1) Pay attention, the Y axis goes from top to bottom (monitor coordinate system)
