Default implementation of instrument margin calculations. Returns the current price of instrument multiplied to quantity as a margin needed Can be used for non-leverage margin calculations for simple cases, when instrument has a cost denominated in BaseCurrency

Examples

Let's suppose base currency is USD, Instrument is AAPL shares, Quantity = 10, Current price = $300, Leverage = 1. Since AAPL is denominated in USD (same to base currency), a margin needed = 300*10/1 = 3000 USD.
