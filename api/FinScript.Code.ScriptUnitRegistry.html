The event triggers when a new script assembly is registered

Property Value

IEvent<AssemblyRegisteredEventArgs>

Remarks

When some script is rebuilt, a new version of script assembly registers and substitutes old version. In this case, the event is raised with RemovedAssemblyInfo not null When an assembly register for the first time, RemovedAssemblyInfo is null
