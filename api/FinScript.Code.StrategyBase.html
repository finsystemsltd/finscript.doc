CalculateMoneyManagementForCurrentBar(TradeAction, OrderType?, double?)

Helper method to calculate MoneyManagement using the close price of last bar from Bars

protected MoneyManagementResult CalculateMoneyManagementForCurrentBar(TradeAction tradeAction, OrderType? orderType = null, double? stopLoss = null)

Parameters

tradeAction TradeAction

A planned trade action (buy or sell) for which we are getting the calculation

orderType OrderType?

Optionally, the type of the planned order (market, stop, limit, etc.). It is recommended to specify the future order type when it is possible, to get more precise calculation

stopLoss double?

Optionally, the stop loss for the new position. It is recommended to always use stop loss to limit your trading risks

Returns

MoneyManagementResult
  1. If MoneyManagement is not set, a zero quantity and text in ErrorMessage
  2. If Bars is empty, a zero quantity and text in ErrorMessage, otherwise, a result of Calculate(MoneyManagementParameters)
