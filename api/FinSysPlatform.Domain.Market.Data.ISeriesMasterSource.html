This is the very basic interface of series. Series is a collection of pairs DateTime, {T} items, where T can be of any type This interface supports only access to DateTime, but not value For full support see ISeries<T>

Remarks

Unlike IList<T> or ICollection<T> series does not support a modification of existing items. In other words, it is impossible to change or delete an already added item (with exception of last item, IsLastItemRemovalSupported it is also impossible to insert item into series (only an adding of items is supported)

Remarks

Any series can only add new items to their end, update last item, and, if this property is true, then delete last item Updating/deleting items inside the collection, as well as inserting items into collection is not supported

Parameters

Time stamp to be converted to an absolute item index

mappingMode TimeToIndexMappingMode

Defines a mode of matching time

Returns

int

Absolute index of item which has the same time to time If there is not exact matching then

  1. if mappingMode is set to ExactMatching then the method returns -1
  2. otherwise nearest left or right index is returned (depending on mode, see details in TimeToIndexMappingMode) Attention: with NearestRight the method may return index in ranges [0..Count] with NearestLeft the method may return index in ranges [-1..Count-1] Always check result for being in ranges [0..Count -1] !
